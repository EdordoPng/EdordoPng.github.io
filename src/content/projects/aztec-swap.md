## Event Context: ETH London 2023

Developed during the high-energy **ETH London Hackathon** (October 2023), organized by **Encode Club**. The event brought together developers at 8 Northumberland Avenue to innovate within the Ethereum ecosystem.
Amidst marathon coding sessions and workshops on **Programmable Privacy**, this project emerged as one of the standout submissions.

Blog Event Page : "https://www.blog.encode.club/eth-london-hackathon-prizewinners-and-summary-85cdff4d093d"

## Project Overview

**AztecSwap** is a Proof-of-Concept for a privacy-preserving Decentralized Exchange (DEX). 
It leverages **Aztec's Noir**, a Domain Specific Language for Zero-Knowledge (ZK) proofs, to enable private token swaps on Ethereum. The goal was to demonstrate how "Programmable Privacy" can be applied to DeFi, allowing users to trade without revealing sensitive transaction data on-chain.
